# SFD — Space for Debate

7 AI specializzate (Cosmo, Hermes, Psiche, Asclepio, Prometeo, Ares, Temide) dibattono su qualsiasi input umano.
La fase visibile mostra ogni risposta in dashboard; la fase invisibile usa RecursiveMAS per comunicazione telepatica nello spazio latente.
Il consenso emerge naturalmente — può essere 7/7 o 1/7, riflette la reale complessità del problema.
Stack: Llama 8B locale via mlx_lm, RecursiveMAS, React + FastAPI, memoria a grafo.
L'umano è sempre al centro e approva l'output finale.
